Tentang Saya

Halo, saya Ilham Maulana, siswa SMK Wikrama Bogor jurusan TJKT. Saat ini saya sedang mempersiapkan diri untuk mengikuti Praktik Kerja Lapangan (PKL) guna meningkatkan kemampuan teknis dan profesional di bidang teknologi informasi. Saya juga sering mempelajari dan mencoba praktik seperti konfigurasi jaringan, penggunaan server Linux, serta dasar cloud computing untuk menambah pengalaman.


Saya pribadi senang belajar hal baru, bisa bekerja sama dengan tim, dan berusaha menyelesaikan tugas dengan baik sesuai kemampuan saya.
